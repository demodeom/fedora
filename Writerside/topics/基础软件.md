# 基础软件

Fedora 43 基础软件包安装与配置指南

## 概述
本文档详细介绍了 Fedora 43 系统中常用基础软件的安装方法及关键配置，涵盖网络工具、Shell 环境、版本控制、多媒体应用等核心组件。

## 常用软件安装

### 基础网络工具
以下工具是系统管理和开发工作的基础组件：

```bash
# 安装基础工具包
sudo dnf install -y curl wget zsh git vim
```

| 软件 | 功能说明 | 主要用途 |
|------|---------|---------|
| **curl** | 命令行 HTTP 请求工具 | API 调试、文件传输、支持多种网络协议 |
| **wget** | 命令行下载工具 | 递归下载、断点续传、支持 HTTP/HTTPS/FTP |
| **zsh** | 功能强大的交互式 Shell | 支持插件系统、主题定制、智能补全 |
| **git** | 分布式版本控制系统 | 代码管理、协作开发、版本追踪 |
| **vim** | 高效文本编辑器 | 代码编辑、配置文件修改、支持插件扩展 |

## 多媒体组件安装

### FFmpeg 完整安装 {id="ffmpeg_1"}
FFmpeg 是功能强大的音视频处理工具集，支持格式转换、流媒体处理等高级功能。

#### 启用 RPM Fusion 仓库
由于 FFmpeg 涉及专利编码格式，需通过第三方仓库安装完整版本：

```bash
# 添加 RPM Fusion free 仓库（开源软件）
sudo dnf install https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm

# 添加 RPM Fusion nonfree 仓库（专有软件，含专利编码格式）
sudo dnf install https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm
```

#### 安装 FFmpeg 及其开发库
```bash
# 更新软件包缓存
sudo dnf update

# 安装 FFmpeg 主程序及开发头文件
# --allowerasing 参数用于解决可能的包冲突
sudo dnf install ffmpeg ffmpeg-devel --allowerasing
```

### 媒体播放器安装
```bash
# 安装图形化媒体播放器
sudo dnf install vlc mpv
```

| 播放器 | 特点 | 适用场景 |
|--------|------|---------|
| **VLC** | 跨平台、格式支持全面 | 日常视频播放、流媒体接收 |
| **MPV** | 轻量级、命令行驱动 | 高级用户、脚本集成、定制化播放 |

## Git 版本控制配置

### 基础配置
完成 Git 安装后，建议进行以下全局配置：

```bash
# 设置默认分支名称（新仓库默认使用 main 分支）
git config --global init.defaultBranch main

# 配置用户身份（提交记录中显示的作者信息）
git config --global user.name "您的用户名"
git config --global user.email "您的邮箱@example.com"

# 解决中文文件名显示问题
# 禁止对非ASCII字符进行转义，直接显示原始中文
git config --global core.quotepath false
```

### 配置说明
| 配置项 | 作用 | 推荐值 |
|--------|------|--------|
| `init.defaultBranch` | 新仓库默认分支名 | main（现代标准） |
| `user.name` | 提交者名称 | 您的实际姓名或昵称 |
| `user.email` | 提交者邮箱 | 与代码托管平台一致的邮箱 |
| `core.quotepath` | 路径显示编码 | false（显示原始字符） |

## Shell 环境增强

### Oh My Zsh 安装
Oh My Zsh 是一个流行的 Zsh 配置管理框架，提供丰富的主题和插件支持：

```bash
# 通过 curl 执行远程安装脚本
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

安装完成后：
1. 默认 Shell 会自动切换为 Zsh
2. 配置文件位于 `~/.zshrc`
3. 可通过编辑该文件选择主题和启用插件

### Zsh 常用配置建议
在 `~/.zshrc` 中可调整以下设置：

```bash
# 设置主题（例如使用 agnoster 主题）
ZSH_THEME="agnoster"

# 启用常用插件
plugins=(git git-flow docker kubectl autojump zsh-autosuggestions)
```

## 验证安装

完成所有安装后，可通过以下命令验证各组件：

```bash
# 验证基础工具版本
curl --version
wget --version
zsh --version
git --version
vim --version

# 验证多媒体组件
ffmpeg -version
vlc --version
mpv --version

# 验证 Git 配置
git config --global --list
```

## 故障排查

### 常见问题处理

1. **RPM Fusion 仓库添加失败**
   - 检查网络连接
   - 确认 Fedora 版本号正确
   - 尝试使用国内镜像源

2. **FFmpeg 安装冲突**
   - 使用 `--allowerasing` 参数解决包冲突
   - 考虑卸载冲突软件包后重试

3. **Oh My Zsh 安装中断**
   - 手动切换 Shell：`chsh -s $(which zsh)`
   - 重新运行安装脚本
